# 📌 Project Overview

The **Virat Kohli Career Stats Dashboard** is an interactive sports analytics dashboard developed using Microsoft Power BI to analyze Virat Kohli’s cricket performance across different formats and years.

The dashboard provides deep insights into batting performance, scoring trends, strike rate analysis, centuries & half-centuries distribution, and boundary statistics using visually engaging analytics and KPI-driven storytelling.

This project demonstrates practical skills in data visualization, sports analytics, dashboard storytelling, and business intelligence using Power BI.

---

# 🚀 Dashboard Features

## ✅ Key Functionalities

* Format-wise Performance Analysis
* Year Range Filtering
* Runs Trend Analysis
* Boundary Distribution Analysis
* Strike Rate Monitoring
* Average Performance Tracking
* 100s & 50s Trend Analysis
* Interactive KPI Dashboard

# 📈 Dashboard Components

## 1️⃣ Runs by Year Analysis

This visualization tracks Virat Kohli’s yearly run performance.

### Key Insights:

* Peak scoring years recorded between 2016–2019.
* Performance fluctuations observed after 2020.
* Consistent run contribution across formats.

---

## 2️⃣ Format-Wise Performance

The dashboard allows filtering performance across:

* ODI
* T20I
* Test

### Business Value:

* Enables comparative performance analysis.
* Helps identify strongest playing format.
* Supports sports analytics storytelling.

---

## 3️⃣ Boundary Analysis

A donut chart visualizes:

* Total 4s
* Total 6s

### Key Insights:

* Boundary scoring is heavily dominated by 4s.
* Aggressive batting patterns contribute to strike rate consistency.

---

## 4️⃣ 100s & 50s Trend Analysis

This section tracks yearly batting milestones.

### Insights:

* High consistency in half-centuries.
* Century conversion rates vary by season.
* Peak performance periods align with highest run years.
